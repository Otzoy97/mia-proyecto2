# GTSales Marketplace
## Descripción
Aplicación web para ejemplificar el uso de Oracle Database Express Edition, NodeJS y Angular.

Se usa Angular como cliente, NodeJS para el servidor y Oracle XE 18 se utiliza como base de datos.
